To see the compiler in action, you can try pasting the content of a Talon example file into the text box and clicking "Compile and run". (You'll need to have your browser's JavaScript console open in order to see the output.)

The "Initial file" dropdown allows you to choose which file (if any) will be automatically run by the compiled script (any others will need to be imported or initiated externally). "Auto" means that the text input will run if it has content, and otherwise the first file from the file input will be run.
